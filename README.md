# LeadOS GitHub Pages

This folder is ready for free GitHub Pages hosting.

## Files

- `index.html` — the full standalone app
- `.nojekyll` — tells GitHub Pages to serve the site as-is

## Free publish steps

1. Create a new GitHub repository.
2. Upload everything from this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and the `/ (root)` folder.
6. Save, then wait for GitHub Pages to publish.

Your site URL will look like:

`https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

## Data note

The app stores data in the browser on each device. Use the in-app backup export/import buttons to move data between devices manually.
